# flow_mcp

仓库根目录存放 **四阶段工程文档**（`ai-software-engineering/`）与 Cursor 配置等；**Vue + Vite 流程图 SDI 应用** 位于子目录 **`flow_vue/`**。

## 快速开始

```bash
cd flow_vue
npm install
python dev.py
```

或在仓库根目录执行（脚本会切换到 `flow_vue` 再执行 npm）：

```bash
python flow_vue/dev.py
```

详见 [`flow_vue/README.md`](flow_vue/README.md)。

# 用户说明书

## 产品形态

流程图 **SDI** 单窗口编辑：主工作区**左侧**为**节点库**，**右侧主列**上方为**预览**、下方为**编辑区**（与实现一致）。**预览**与编辑区同为 **4:3** 逻辑比例，并在可用空间内自适应缩放；预览画布上在对应虚线框旁直接标注 **4:3**、**16:9**、**1:1**，便于构图。主界面采用 **Windows 风格**标题栏与菜单栏（**文件 / 语言 / 主题 / 帮助**）；其下为**工具栏**（运行控制、**全屏**）。预览与节点库之间的**竖向分割条**、上区与编辑区之间的**横向分割条**均可拖动；**底部输出区**默认**折叠**，展开后与主区之间有**横向分割条**可调高度。右侧为**节点属性** Dock（缘条 + 显示区）。界面正文默认使用较大字号与行距，便于阅读。

## 工具栏与全屏

- **运行 / 暂停 / 停止**：以图标形式居中显示。
- **全屏**（工具栏右侧）：将本应用整页以浏览器全屏方式铺满显示器（仍保留系统退出全屏方式，如 `Esc`，依浏览器而定）。
- **快捷键**：**Ctrl + 空格** 切换全屏（在输入框、文本域等聚焦时不会触发）。在 macOS 上 **⌘ + 空格** 常被系统占用，请优先使用工具栏按钮或 **Ctrl + 空格**（若浏览器未拦截）。

## 底部输出区

- 与 **状态栏上的日志（Log）** 不同：输出区用于**流程/节点等业务侧**可读的文本输出（如脚本打印）；默认**折叠**，点击底部横条 **展开输出** 后显示。
- 输出区内可复制全文、清空；可拖动输出区**上边缘**调整高度。

## 页内最大化（预览或编辑）

- 在 **预览** 或 **流程图编辑** 标题栏右侧使用最大化按钮，可仅保留菜单、工具栏与当前区域（隐藏节点库、属性区、状态栏等）；再次点击恢复。

## 语言与主题

- **语言**：主菜单栏 **「语言」** 顶层项下切换中文 / English。
- **主题**：主菜单栏 **「主题」** 顶层项下切换跟随系统 / 浅色 / 深色。

## 状态栏与日志

- **状态栏**（窗口底部）：显示就绪信息与最近一条摘要。
- **点击状态栏**：打开 **Log 窗口**，可查看历史日志（纯文本）。
- **复制**：在 Log 窗口使用复制按钮，将当前全文复制到剪贴板。

## 编辑与文件

- **编辑画布**：含**背景网格**与过画布几何中心（800×600 逻辑区之半宽/半高）的**十字中心线**；**鼠标中键拖动**平移视图，**滚轮**以指针处为中心缩放。
- 在**节点库**选择节点类型，在**编辑区**生成节点并拖动调整位置。
- **连线**（若已启用）：按界面提示先选起点节点再选终点节点。
- **文件**（菜单「文件」）：**新建**（清空当前图，未保存时会提示）、**打开**（选择 `.json` 流程图文件）、**保存**（若已通过系统对话框获得可写文件句柄则直接写入；否则行为与另存为类似）、**另存为**（指定文件名保存 JSON）。标题栏显示当前文档名；有未保存更改时，新建/打开会提示确认。

## 常见问题

- **页面空白**：确认使用现代浏览器并允许运行本地脚本；开发态需先执行依赖安装与启动命令（见 `developer-manual.md`）。

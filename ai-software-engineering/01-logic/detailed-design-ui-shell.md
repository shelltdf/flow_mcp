# 详细设计：页内壳层与布局

## 布局结构

1. **根容器**：纵向 flex；**菜单栏**顶层为 **文件、语言、主题、帮助**（Windows 风格下拉）。
2. **工具栏**（菜单栏下方）：**运行 / 暂停 / 停止**（图标）；**右侧**为 **全屏**（调用浏览器 Fullscreen API，使整个文档根元素全屏显示）；快捷键 **Ctrl+空格** 切换全屏（在输入类控件聚焦时不拦截）。
3. **工作区栈 `workspace-stack`**（flex:1）：  
   - **主带 `main.workspace-shell`**：横向为 **左侧节点库缘条 + 节点库**（缘条可折叠整库；标题栏折叠/最大化）、**中间主列**（上：预览；下：编辑；中间横向分割条）、**右侧属性 Dock 与缘条**（与既有 Dock 规范一致）。  
   - **底部输出区**：默认**折叠**，展开后为 **输出 Dock**（纯文本、等宽字体），与主带之间有**横向分割条**可调高度；供节点/运行时代码通过 `consolePrint` 写入，与 **诊断 Log**（状态栏 → Log 弹窗）职责分离。
4. **主列内上下分栏**（位于节点库右侧）：**上**为 **预览区**，**下**为 **编辑区**，中间为横向分割条。
   - **预览区**：展示当前图的缩小视图；参考比例 **4:3 / 16:9 / 1:1** 在预览 SVG 内以标签标在对应虚线框旁。
   - **编辑区**：画布左上角 **快捷键** 按钮，悬停列出说明。
5. **页内全窗口聚焦**：预览或编辑标题栏 **最大化** 时，仅保留菜单栏、工具栏与当前 **预览或编辑** 之一，隐藏其余壳层；退出后恢复布局快照。
6. **右侧 Dock（节点属性）**：与主工作区并列，靠窗口最右为**缘条**（展开/折叠）；展开时为**显示区**，含标题栏（折叠、最大化）与属性表单；与主区之间有**纵向分割条**可调宽度。
7. **状态栏**：主工作区栈下方一条（全窗口聚焦模式下可隐藏）；点击打开 **Log** 弹层（诊断日志，与底部 **输出** 不同）。

## 状态与消息

- **诊断日志（appLog）**：应用内统一追加字符串行，带级别；状态栏可只显示最近一条摘要；Log 弹层展示历史。
- **节点输出（nodeConsole）**：仅流向底部 **输出 Dock**；由 `consolePrint` / `consoleClear` 等维护，供业务扩展。
- **主题/语言**：与 `prefers-color-scheme` 或用户覆盖一致时，写入偏好存储（若启用）。

## 与 IDE 宿主关系

若在 **VS Code Webview** 等内嵌运行，IDE 侧栏与本应用页内分区**独立**；本文档描述页内可见区域。

# 详细设计：页内壳层与布局

## 布局结构

1. **根容器**：纵向 flex；**菜单栏**顶层为 **文件、语言、主题、帮助**（Windows 风格下拉）。
2. **上条带**：横向 flex，高度可固定或比例分配。
   - **预览区**（左）：展示当前图的缩小视图或视口框；依赖流程图模型的只读订阅。
   - **节点库区**（右）：列表或分组；用户选择类型后，在编辑区创建节点（点击或拖放，以实现为准）。
3. **编辑区**：占据上条带下方剩余空间；捕获主要指针/键盘交互。
4. **右侧 Dock（节点属性）**：与主工作区并列，靠窗口最右为**缘条**（展开/折叠）；展开时为**显示区**，含标题栏（折叠、最大化）与属性表单；与主区之间有**纵向分割条**可调宽度。
5. **状态栏**：底部一条；点击打开 Log 弹层。

## 状态与消息

- **全局日志**：应用内统一追加字符串行，带级别；状态栏可只显示最近一条摘要。
- **主题/语言**：与 `prefers-color-scheme` 或用户覆盖一致时，写入偏好存储（若启用）。

## 与 IDE 宿主关系

若在 **VS Code Webview** 等内嵌运行，IDE 侧栏与本应用页内分区**独立**；本文档描述页内可见区域。

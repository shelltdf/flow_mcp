# 接口设计（程序间）

> 本文**仅**描述程序间数据交换形态，**不包含** GUI 操作、快捷键与窗口消息。

## 流程图文档文件 / 交换（JSON）

| 项目 | 说明 |
|------|------|
| **用途** | **新建 / 打开 / 保存 / 另存为** 均针对同一套流程图 JSON；供本应用与外部工具交换。 |
| **编码** | UTF-8 JSON。 |
| **建议扩展名** | `.json` 或 `.flow.json`。 |
| **顶层字段（初版）** | `version: 1`、`nodes`（数组）、`edges`（数组）；字段以 `02-physical/flowchart-sdi/spec.md` 为准。 |
| **浏览器持久化** | 支持 **File System Access** 时可将「保存」写回用户所选文件句柄；否则以浏览器下载或系统「另存为」对话框为准。 |

## HTTP / gRPC

当前无可对外服务端；标注为**不适用**。若增加 REST API，在此扩展并同步物理规格。

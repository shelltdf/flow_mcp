# 软件设计

## 与子系统划分

| 子系统 | 职责 |
|--------|------|
| **壳层与布局** | 根布局：菜单 + **工具栏**（运行控制、**浏览器全屏**）+ **工作区栈**（主区：上条带「预览 \| 节点库」+ 下编辑区；**底部输出 Dock** 可折叠）；主题/语言入口；**状态栏与诊断 Log**；页内全窗口聚焦（仅预览或仅编辑）。 |
| **节点输出** | 底部 **输出** 区（`consolePrint` 等），与诊断 Log 分离，供节点/运行时写入可读文本。 |
| **流程图模型** | 节点、边、画布坐标、选中与撤销栈（演进中）。 |
| **预览** | 订阅模型变化，缩略展示或缩放预览。 |
| **节点库** | 节点类型列表、分组、向编辑区添加实例的入口。 |
| **持久化（可选）** | 本地存储或导出文件；见 `database-design.md` 与 `interface-design.md`。 |

## 构建目标对应

主交付物为 **`flowchart-sdi`**（见 `02-physical/README.md`）：Vue + Vite 单页应用，源码位于 `flow_vue/`（与 `ai-software-engineering/` 分离）。

## 技术约束

- **前端框架**：Vue 3（组合式 API 推荐）。
- **路径与文件名**：实现侧 ASCII。
